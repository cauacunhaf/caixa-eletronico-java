## Sistema de Caixa Eletrônico em Java

Projeto desenvolvido em Java com o objetivo de praticar programação orientada a objetos (POO).

## 📌 Descrição

Este projeto simula o funcionamento básico de um caixa eletrônico, permitindo que o usuário interaja com uma conta bancária através do terminal.

O sistema foi criado para praticar conceitos fundamentais da programação orientada a objetos utilizando a linguagem Java.

## ⚙️ Funcionalidades

O sistema permite:

- Criar contas
- Consultar saldo
- Realizar depósitos
- Realizar saques
- Listar contas cadastradas

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos
- Estruturas de decisão e repetição
